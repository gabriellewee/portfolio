  • Details

    My Passionate Guild Romance is a dating role-playing game set in medieval times. Players are eligible bachelors or bachelorettes who have the choice of playing as a mage, ranger, knight, or brawler. Not only must players romance the character of their choice, but must also save a guild on the verge of disbanding!

    This project was a collaboration between several students. I took the role of UI Designer - responsibilities included creating a logo, most interface elements, putting together menus and buttons in Flash, and designing documentation for the project. Teammates provided artwork, character elements, and programming.

    Documentation, downloadable here, includes a detailed breakdown of user interface and explanation of game mechanics. I designed the document and acted as editor for all writing.
