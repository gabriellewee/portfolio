About

I'm a twenty-three year old web developer who is currently in the process of majoring in Game Art & Design at Ex'pression College for Digital Arts. My personal blog and portfolio resides here.

When I initially bought gabriellew.ee I was looking for a static blog generator that had a front-end CMS, but wasn't finding anything that fit my specifications. I started out on Wordpress and found it far too bloated and unwielding for the simple blog I was planning on. I tried several different alternatives, including Scriptogr.am , and finally decided to bite the bullet and try Jekyll. Completely worth all the frustration spent trying to wrap my brain around the programming.

This site is generated using Jekyll and manually pushed to Github via command line which hooks into the domain. I have basic hosting from Hostgator which doesn't allow me to directly install Jekyll on my website. I host all my files in Dropbox for easy access - I can update files at any time and push the changes to Github when I get back home to my computer.

If you have any questions about how I set this up, please feel free to contact me using any of the links on the contact page.
